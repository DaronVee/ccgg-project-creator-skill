## Use It (Just Ask)

Say any of these:

- "Create new project"
- "Initialize new incubator project"
- "Set up project structure for [your-project-name]"

Claude will ask you a few questions and create everything automatically.

## What It Creates For You

Every project includes:

1. **CLAUDE.md** - Project guidance with Parent System Integration
2. **README.md** - Quick start guide
3. **Active Projects Index Entry** - For cross-project searches
4. **PHASE_TRACKER.md** (if multi-phase) - Phase management
5. **Coordination Hub/** (if complex) - Dependency management

Time saved: **15-30 minutes per project**
